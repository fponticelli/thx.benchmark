package thx.macro

class MacroFields

Available on neko

Extension methods to work with class fields at macro time.

Static methods

staticgetFirstMetaParam (field:Field, name:String):Expr

Extracts the first meta entry parameter for name. It returns null if it does not exist.

staticgetMetaEntry (field:Field, name:String):Null<MetadataEntry>

Returns a MetaEntry with name if it exists or null otherwise.

staticgetMetaParams (field:Field, name:String):Array<Expr>

Returns the params of a meta with name. It always returns an array even if no matches are found.

staticgetMetaParamsAsStrings (field:Field, name:String):Array<String>

Returns the params as Array<String> of a meta with name. It always returns an array even if no matches are found.

staticgetType (field:Field):ComplexType

Returns the type of field.

statichasMeta (field:Field, name:String):Bool

Returns true if the field has a meta with the supplied name.

staticisInstance (field:Field):Bool

Returns true if the field has NOT the static qualifier.

staticisMethod (field:Field):Bool

Returns true if the field is a method.

staticisProperty (field:Field):Bool

Returns true if the field is a property.

staticisPublic (field:Field):Bool

Returns true if the field has the public qualifier.

staticisStatic (field:Field):Bool

Returns true if the field has the static qualifier.

staticisVar (field:Field):Bool

Returns true if the field is a variable.
