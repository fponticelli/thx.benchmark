package thx.macro

class MacroTypes

Available on neko

Extension methods to work with types at macro time.

Static methods

staticargumentIsOptional (fun:Type, index:Int):Bool

It returns true if the argument at position index is optional for the given function.

staticbaseTypeToTypePath (t:BaseType, params:Array<TypeParam>):TypePath

staticgetArgumentType (fun:Type, index:Int):Type

It returns the type of an argument at index for a given function.

staticgetArity (fun:Type):Int

It returns the arity(number of arguments) of a given function.

staticgetClassTypeParameters (type:Type):Array<Type>

It returns an array of type parameters.

staticgetFunctionArgument (fun:Type, index:Int):{t:Type, opt:Bool, name:String}

It returns the argument description for a function at index.

staticgetFunctionArguments (fun:Type):Array<{t:Type, opt:Bool, name:String}>

It returns an array of argument descriptions for a given function.

staticgetFunctionReturn (type:Type):Type

It gets the return type of a function.

staticisFunction (type:Type):Bool

It returns true if the passed type is a function.

staticqualifyComplexType (ct:ComplexType):ComplexType

Fully-qualify a ComplexType. For example, turn Option<Int> to haxe.ds.Option.Option<StdTypes.Int> when haxe.ds.Option is in context view using or import. If the type cannot be qualified the original ComplexType is returned.

staticqualifyType (t:Type):ComplexType

statictypeInheritance (type:Type):Array<String>

It returns an array of types in string format representing the inheritance chain of the passed Type.

statictypeToTypePath (t:Type):TypePath
